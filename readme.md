# Webcam Catcher

Work with OpenCV

This project can:

* Stream from a webcam on JFrame
```
java -jar wc.jar
```
* Record video from webcam to file
```
java -jar wc.jar -r
```
* Take a photo from a webcam
```
java -jar wc.jar -cap
```
* Stream from a webcam to localhost
```
java -jar wc.jar -stream
```
